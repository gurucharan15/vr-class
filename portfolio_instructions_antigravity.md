## Tech Stack Constraints
- **HTML5:** Semantic architecture.
- **CSS3:** Advanced styling, keyframe animations.
- **JavaScript (ES6+):** Core logic, 3D animations (utilizing WebGL/Three.js via CDN to meet the "Highly animated 3D" requirement within the JS scope).
- **Bootstrap:** Responsive grid, layout structuring, and utility classes.

## Design System: "Liquid Glass"
The core aesthetic is **Liquid Glass**. 
- **Visuals:** Translucency, frosted glassmorphism, dynamic refraction, and smooth, flowing liquid-like morphing shapes in the background.
- **Colors:** Professional corporate palette (deep blues, clean whites, subtle metallic accents) enhanced by light dispersion effects passing through the "glass" elements.
- **Interactions:** Hover states should ripple or distort like water/glass. 3D objects should react to mouse movement (parallax) and scroll position.
